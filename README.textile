h1. Wygwam No Upload Tab extension

This EE2 extension removes the “Upload” tabs from the Link, Image, and Flash dialogs, forcing authors to click the “Browse Server” buttons and upload files via CKFinder.

h3. Requirements

* "ExpressionEngine":http://expressionengine.com/index.php?affiliate=brandonkelly&page=/public_beta/ 2.0.1 build 20100215 or later
* "Wygwam":http://pixelandtonic.com 2.0.1 or later

h3. Installation

* Create a folder called “wygwam_no_upload_tab” in system/expressionengine/third_party/
* Upload ext.wygwam_no_upload_tab.php to system/expressionengine/third_party/wygwam_no_upload_tab/
* Enable Wygwam No Upload Tab in Add-Ons → Extensions
